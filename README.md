Если хочешь аргумент от бота то напиши ему "Ты тупой".
Если хочешь случайный пароль из 12 символов то пропиши боту "/password".
Если хочешь случайный эмоджи (их 4) то пропиши боту "/emodji".
Если хочешь первые три одинаковых эмоджи (не случайных) то пропиши /e1
Если хочешь вторые три одинаковых эмоджи (не случайных) то пропиши /e2
Если хочешь третие  три одинаковых эмоджи (не случайных) то пропиши /e3
